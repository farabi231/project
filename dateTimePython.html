Python Datetime



Dates in Python can be defined by using the built in datetime library, this module imports numerous class methods and functions which allows developers to manipulate time data with ease, this is especially useful if users of Python is working with timeseries datatypes and want to make comparisons easily.



One key feature of the module is that it also makes it easy working with Unix timestamps. The Unix timestamp is a measure of time which displays the number of seconds passed from a predetermined date, the predetermined date is currently set to the 1st of January 1970, the format is popular because it converts dates to an integer and is widely compatible by coding languages of different kind.



Another key feature of the datetime module is that it allows converting text in string format to datetime values. Often, humans note down dates in a human readable format, however, this format is not readable by code and is difficult to represent it digitally. The datetime module introduces the .strptime() function, which reads human readable dates and converts it to a datetime object, this object can then be easily be formatted by other functions within the module.
