Python Variables



Python variables are an essential part of the Python programming language. They are used to store values and data types within a Python program, and allow developers to manipulate and use these values throughout the program.



A Python variable is essentially an object that is used to refer to a value or data type. These names are created by assigning a value to a keyword using the equal operator(‘=’). Once a variable has been created, it can be used throughout the program to access and modify the value it contains. Unlike many other coding languages, Python variables are dynamically typed, which means they do not need to be declared before use.



One of the key features of Python variables is their ability to store different data types, some of the most common ones are as follows:

  • Numbers: Integer & floats
  • Strings: Stores word characters
  • Booleans: True or False
  • Lists: Where items are stored in order of there time added
  • Tuples: Same properties as list but is immutable and items can’t be removed
  • Dictionaries: Variables can be assigned to keys


Besides simple data types, Python variables can also contain intricate entities like classes, functions, and modules. This enables programmers to produce code that is exceptionally modular and reusable, which can be conveniently distributed and adjusted for diverse projects.



Similar to JavaScript, Python variables can contain letters, numbers or even underscores, however, they may not begin with numbers as the interpreter within Python will not be able to distinguish it with regular number values. It is to be noted that variable names in Python are case-sensitive, this means that variable names such as ‘var1’ and ‘Var1’ represents two completely different items in the memory space. On the contrary, since Python is a high level language, it supports one line value swapping statements(eg. a, b = b, a) which is not possible in traditional coding languages using JavaScript
