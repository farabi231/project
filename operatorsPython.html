Python Operators



Python provides built-in expressions, known as operators, that allow for performing arithmetic operations and basic logical operations on specified values and variables. By using these operators, users can easily perform mathematical calculations and manipulate data in their programs.

Other operators includes:

  1. Arithmetic Operators
    • ‘+’ Addition Operator
    • ‘-‘ Subtraction
    • ‘*’ Multiplication
    • ‘/’ Division
    • ‘//’ Floor division Operator
    • ‘%’ Modulus Operator
    • ‘**’ Exponentiation Operator
  2. Arithmetic Operators
    • ‘=’ Equal
    • ‘+=’ Add Equals
    • ‘-=’ Minus Equals
    • ‘*=’ Multiply Equal
    • ‘/=’ Division Equal
    • ‘%=’ Modulus Equal
    • ‘//=’ Floor Division Equals
    • ‘**=’ Exponent Equals
  3. Comparison Operators
    • ‘==’ Equality
    • ‘!=’ Not Equals
    • ‘>’ Greater Than
    • ‘<' Less Than
    • ‘>=’ Greater or equal to
    • ‘<=’ Less Than for equal to

Python, like any programming language, relies on operators as a fundamental building block. These operators make it possible to accurately manipulate and compare data using mathematical and logical operations.
